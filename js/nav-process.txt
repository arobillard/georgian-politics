Toggle:
- on Open:
  - changes toggle from hamburger to X
  - opens nav-wrap
- on Close
  - closes Level 2, Level 1, and nav-wrap
  - changes from X to hamburger

Main Nav /w drop down:
- when closed:
  - closes all other open dropdowns in order from lowest to highest
  - opens immediate child dropdown
  - adds selected highlighting
- when open:
  - closes all child dropdowns in order from lowest to highest
  - removes selected highlighting

Level 1 Drop down /w dropdown:
- when closed:
  - closes all child dropdowns of siblings and siblings
  - opens immediate child dropdown
  - adds selected highlighting
- when open:
  - closes all child dropdowns
  - removes selected highlighting
